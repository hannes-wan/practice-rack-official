# Practice Rack

A fast-launching desktop guitar practice rack with tuner, looper, recorder, and effects.

Product page: https://hannes-software.com/practice-rack/
Downloads: https://hannes-software.com/practice-rack/#downloads
Latest release: https://github.com/hannes-wan/practice-rack-official/releases/tag/v0.1.4
