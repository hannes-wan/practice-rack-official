# Practice Rack 0.1.7

**Practice Rack Community Edition is free to download and use.** This release keeps the core desktop workflow available without a subscription requirement.

## What stands out in this release

- The borderless rack window is now natively resizable while preserving the supported minimum workspace geometry.
- Hardened macOS store packaging with the product microphone purpose string, Music category, minimum-system target, and sandbox metadata.
- Corrected optional-feature copy: the free Community Edition keeps the complete effects rack, while the licensed boundary is limited to saved-preset capacity and the registered sync integration.

## Choose the right package

- **Linux x64 (APPIMAGE):** `practice-rack-0.1.7-linux-x64.AppImage` — Portable Linux desktop package
- **Linux x64 (DEB):** `practice-rack-0.1.7-linux-x64.deb` — Debian, Ubuntu, Mint, and compatible systems
- **Linux x64 (RPM):** `practice-rack-0.1.7-linux-x64.rpm` — Fedora, RHEL, openSUSE, and compatible systems
- **Windows x64:** `practice-rack-0.1.7-windows-x64-setup.exe` — Guided Windows installer

## Before you install

- Instrument input is processed in the local audio path; recorded takes and loaded impulse responses stay on the desktop unless you share them.
- Keep the installer filename and checksum together when handing a package to another machine.
- Read the [help center](https://hannes-software.com/practice-rack/help/) for supported inputs, workflow boundaries, and troubleshooting.
- Optional licensed features are described on the website without changing the free Community Edition download.
